# **🔗 URL Shortener API**  

<p>This project is a URL Shortening service built with <b>Node.js, Express, and MongoDB</b>. It allows users to shorten long URLs and retrieve the original links.</p>

## **🚀 Technologies Used**  
<ul>
  <li><b>Node.js & Express.js</b> – Backend framework</li>
  <li><b>MongoDB</b> – Database for storing short and original URLs</li>
  <li><b>shortid</b> – Generates unique short URLs</li>
  <li><b>JWT</b> – User authentication </li>
</ul>
